============
Installation
============

.. contents:: Table of contents
    :depth: 1
    :local:

1. Install GDAL
---------------

One of the easiest ways to install GDAL is to use `Anaconda <https://www.anaconda.com/>`_

Then simply

``conda install gdal``

For other methods see the `GDAL documentation <https://gdal.org/>`_

2. Install Dask Mosaic from the Github repo.
--------------------------------------------

``pip install git+https://github.com/bluegeo/dask-mosaic``
